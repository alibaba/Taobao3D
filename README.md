# Introduction
We are the TaoTian Meta Technology Team（[淘天Meta技术团队](https://alibaba.github.io/Taobao3D)）, dedicated to exploring and implementing 3D/XR technologies. Our research focuses on 3D object reconstruction, 3D human reconstruction / editing / driving, cross-platform 3D rendering engines, on-device AI inference engines（[MNN](https://github.com/alibaba/MNN)）, and cross-platform XR R&D solutions.
